## 开始

```bash

git clone --depth 1 --single-branch https://github.com/zhouzuchuan/vue-boilerplate.git your-project-name

cd your-project-name

npm install
```

### 开发环境 编译和 hot-reloads

```bash
npm run start
```

### 生产环境 编译并最小化

```bash
npm run build
```

### 启动测试

```bash
npm run test
```

### 启动检查

```bash
npm run lint
```

查看 [官方说明](https://cli.vuejs.org/config/).
